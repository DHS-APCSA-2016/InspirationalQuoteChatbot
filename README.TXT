PROJECT TITLE: Inspirational Quote Chatbot
PURPOSE: Create a chatbot that responds to users through inspirational quotes.
CLASSES: Chatbot and Runner 
MAIN METHODS: 
printStatement() - creates the effect that the chatbot is typing
getGreeting() - starts the conversation off with a greeting by the chatbot
getResponse() - forms a response based off of the user input
findKeyword() - helps filter through possible responses after searching for a keyword

VERSION or DATE: November 2016
AUTHORS: Justin Symmank and Justine Koa
GETTING STARTED: Type messages to the chatbot to have a conversation and respond "Bye" when done!
